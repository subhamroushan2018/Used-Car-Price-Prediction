# 🤖 Model Documentation

## Objective

Build a regression model capable of predicting the selling price of a used car based on its features.

---

## Machine Learning Algorithms Used

The notebook compares multiple regression algorithms:

- Linear Regression
- Lasso Regression
- Ridge Regression
- Support Vector Regressor (SVR)
- Decision Tree Regressor
- Random Forest Regressor

---

## Workflow

1. Load Dataset
2. Data Cleaning
3. Feature Encoding
4. Feature Scaling
5. Train-Test Split
6. Model Training
7. Hyperparameter Tuning using RandomizedSearchCV
8. Model Evaluation

---

## Evaluation Metrics

The models are evaluated using:

- Mean Absolute Error (MAE)
- Mean Squared Error (MSE)
- R² Score

---

## Libraries Used

- scikit-learn
- pandas
- numpy
- matplotlib
- seaborn

---

## Output

The trained regression model predicts the estimated selling price of a used car based on the provided input features.

The notebook also compares the performance of different regression algorithms to identify the best-performing model.
